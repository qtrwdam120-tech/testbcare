# Dashboard Scaffold

هذا مجلد يحتوي على هيكل أولي للوحة إدارة منطقية للمشروع.

## الملفات
- index.html: الصفحة الرئيسية للوحة الإدارة
- styles.css: التنسيق والواجهة
- app.js: بيانات تجريبية لعرض الطلبات والإحصائيات

## ما الذي سيتم إضافته لاحقًا
- ربط البيانات الحقيقية من Backend/API
- نظام تسجيل دخول للمشرفين
- صفحات تفاصيل الطلب
- الرسائل والتحديثات في الوقت الحقيقي عبر Socket.io
- إدارة التوجيه والتغييرات على حالة العميل
